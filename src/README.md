# Java MySQL JDBC Connection

This project demonstrates how to **connect a Java application with a MySQL database** using JDBC.

It is a beginner-friendly example to test database connectivity before performing CRUD operations.

---

## Technologies Used

- Java
- JDBC
- MySQL

---

## What This Program Does

1. Loads the MySQL JDBC Driver
2. Connects to a MySQL database
3. Prints connection success or failure message
4. Closes the database connection safely
